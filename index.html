How to install Arch linux by Lorence I.(xnb666)

 

Note: Before installing Arch linux think carefully before doing the installation because everything on your hard drive or whatever storage you are using now will be erased.

 

Arch linux is the part of the most hard operating system on linux! Arch develop in Marh 2002 Linux Family tree

 

1st Download the Arch linux iso Download

Download Rufus Download

Flash the ISO using Rufus, don't change the default by rufus.

Now boot using the USB

Setup Arch

1. type: iwctl (to connect to the internet if you're using a pc no need)

2. type: ping google.com (to check if your PC connection is stable)

3. type: station wlan0 connect "yourwifiname"

3.1 type:your wifi password

4. type: pacman-key --init

5. type: pacman-key --populate archlinux

6. type: bash <(curl -L archtitus.com/archtitus)

pic the optional: ssd (yes) even of not, extf4, server only. else you choose.

That will automatic setup the Arch, input the things needed to install arch such as the account and the language and hard disk for default use : ssd/exfat4/english

when it's done installing remove the usb and type: reboot

AFTER REBOOT

------CONNECT TO WIFI-----

sudo nmcli dev wifi connect wifiname password "wifipassword"

------DOWNLOAD NEEDED PACKAGE------

sudo pacman -S nano vim git unzip less wget curl neofetch htop

------INSTALL YAY AND PARU------

------YAY------

sudo pacman -Syu

sudo pacman -S --needed base-devel git

git clone https://aur.archlinux.org/yay.git

cd yay

makepkg -si

------PARU------

sudo pacman -Syu

sudo pacman -S --needed base-devel git

git clone https://github.com/Morganamilo/paru

cd paru

makepkg -si

sudo git clone https://github.com/prasanthrangan/hyprdots Hyprdots

cd Hyprdots/Scripts/

chmod +x ./install.sh

./install.sh
